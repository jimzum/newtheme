== Description ==

Want to replace the old ← Older posts | Newer posts → links with some page links?

This plugin adds a more styling options to Wp-PageNavi WordPress plugin OR  the_posts_pagination(); WordPress navigation function.

To Use this plugin you must have [Wp Pagenavi](https://wordpress.org/plugins/wp-pagenavi/)  plugin installed on your wordpress blog

**OR**

**If you are not using WP Pagenavi plugin** then use <code><?php the_posts_pagination(); ?></code> WordPress navigation function and our plugin will help you to give a better design.

[View Here](https://www.wponlinesupport.com/wordpress-plugin-addon/styles-wp-pagenavi-addon/) for more details.

= Features include =

* 5+ styles
* Colors change options 
* Use it with Wp-PageNavi WordPress plugin or <code><?php the_posts_pagination(); ?></code> WordPress navigation function and our plugin will help you to give a better design.


== Installation ==

1. Upload the 'styles-for-wp-pagenavi-addon' folder to the '/wp-content/plugins/' directory.
2. Activate the "styles-for-wp-pagenavi-addon" list plugin through the 'Plugins' menu in WordPress.
3. Go to WPOS - Pagenavi menu tab in the left side(WordPress admin menu) and select your settings
